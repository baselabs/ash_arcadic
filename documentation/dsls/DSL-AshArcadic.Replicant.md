# AshArcadic.Replicant

table}` this resource mirrors.

Add it to a host graph resource:

      domain: MyApp.Domain,
      data_layer: AshArcadic.DataLayer,
      extensions: [AshArcadic.Replicant]

    replicant do
      source_table "orders"
      tenant_attribute :org_id
      skip [:internal_notes]
    end
